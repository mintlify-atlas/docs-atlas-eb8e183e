## Overview

`@kreisler/bot-wa-ts` is a powerful TypeScript framework for building WhatsApp bots using the Baileys library. It provides a clean API for handling messages, media, stickers, and more.

**Package**: `@kreisler/bot-wa-ts`  
**Version**: `1.0.0`  
**License**: MIT

## Installation

```bash
npm install @kreisler/bot-wa-ts
```

## Quick Start

```typescript
import { Whatsapp } from '@kreisler/bot-wa-ts';

const bot = new Whatsapp();

await bot.WAConnect();
await bot.loadHandlers();
await bot.loadCommands();
```

##### sendMsgGroup()

Sends multiple messages to one or more chats with batch processing.

<ParamField path="jids" type="string | string[]" required>
  WhatsApp JID(s) to send messages to (e.g., "1234567890@s.whatsapp.net").
</ParamField>

<ParamField path="media" type="AnyMessageContent[] | Promise<AnyMessageContent[]>" required>
  Array of message content to send.
</ParamField>

<ParamField path="options" type="MiscMessageGenerationOptions">
  Additional message options.
</ParamField>

<ParamField path="batchSize" type="number" default="10">
  Number of messages to send per batch.
</ParamField>

<ResponseField name="return" type="Promise<(proto.WebMessageInfo | undefined)[]>">
  Array of sent message info objects.
</ResponseField>

```typescript
const messages = [
  { text: 'Hello!' },

await bot.sendMsgGroup(
  '1234567890@s.whatsapp.net',

);
```

## Usage Examples

### Basic Bot Setup

```typescript
import { Whatsapp } from '@kreisler/bot-wa-ts';

const bot = new Whatsapp();

await bot.WAConnect();
await bot.loadHandlers();
await bot.loadCommands();

console.log('Bot is ready!');
```

### Creating a Command

```typescript
import { CommandImport, ContextMsg } from '@kreisler/bot-wa-ts';

const pingCommand: CommandImport = {
  active: true,
  ExpReg: /^\.ping$/i,
  cmd: async (client, context, match) => {
    const { msg } = context;
    const start = Date.now();
    
    const sent = await msg.reply({ text: 'Pinging...' });
    const latency = Date.now() - start;
    
    await client.sock.sendMessage(
      sent.key.remoteJid!,
      { text: `Pong! ${latency}ms`, edit: sent.key }
    );
  }
};

export default pingCommand;
```

### Handling Media Messages

```typescript
bot.sock.ev.on('messages.upsert', async ({ messages }) => {
  for (const m of messages) {
    const message = new Message(bot, m);
    
    if (message.hasMedia && message.type === 'imageMessage') {
      const media = await message.downloadMediaV2();
      
      if (media) {
        await message.reply({
          text: `Received image: ${media.fileType?.ext}`
        });
      }
    }
  }
});
```

### Creating Stickers

```typescript
const imageBuffer = /* get image buffer */;
const stickerBuffer = await bot.stickerGenerator(imageBuffer);

await bot.sock.sendMessage(jid, {
  sticker: stickerBuffer
```

### Batch Sending Messages
